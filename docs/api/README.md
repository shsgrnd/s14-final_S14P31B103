# API 문서 (API)

GitCat 서비스의 기능 명세 및 인터페이스 명세서를 관리하는 폴더입니다.

**현재 포함된 파일 목록:**
- `01_functional_spec.csv`: 기능 명세서 (CSV 포맷)
- `01_functional_spec.docx`: 기능 명세서 (Word 문서 포맷)
- `01_functional_spec.md`: 기능 명세서 (Markdown 포맷)
- `05_internel_interface_spec.csv`: 내부 인터페이스 명세서 (CSV 포맷)
- `05_internel_interface_spec.md`: 내부 인터페이스 명세서 (Markdown 포맷)
