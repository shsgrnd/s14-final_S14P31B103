## 최종 점검 체크리스트

- [ ]  자동 스냅샷이 2~3초 이내 생성되는가
- [ ]  스냅샷 그룹핑이 적용되는가
- [ ]  스냅샷이 10개 초과 시 자동 삭제되는가
- [ ]  스냅샷 즐겨찾기 표시가 정상 동작하는가
- [ ]  가장 최근 스냅샷만 삭제 가능한가
- [ ]  선택한 과거 스냅샷으로 정상 원복되는가
- [ ]  원복 전 현재 상태가 안전 스냅샷으로 적재되는가
- [ ]  병합 전 충돌 후보 탐지가 정상 동작하는가
- [ ]  충돌 없음 케이스도 정확히 구분되는가
- [ ]  AI 병합 초안 및 중재안이 자동 호출되는가
- [ ]  AI 응답 구조 검증이 수행되는가
- [ ]  병합 후 잔여 충돌 마커 자동 감지가 가능한가
- [ ]  AI 재제안이 정상 제공되는가
- [ ]  변경 기록이 스냅샷과 연동되는가
- [ ]  Accept 시 병합 완료가 반영되는가
- [ ]  사용자 수정 후 반영이 가능한가
- [ ]  Reject 시 미반영 상태가 유지되는가
- [ ]  커밋 메시지 / 브랜치명 / PR / 작업 설명 추천이 가능한가
- [ ]  로컬 브랜치 삭제 실행이 가능한가
- [ ]  미사용 브랜치 자동 감지가 가능한가
</aside>
