# S-09

기능: AI 병합
비고: 자동 호출, Zod 검증 포함
시나리오명: AI 기반 병합 초안 및 중재안 생성
유형: 정상

### 목적

충돌 후보에 대해 AI 병합 초안과 중재안이 자동 생성되고, 결과 구조가 검증되는지 확인한다.

### 사전 상태

- 충돌 후보가 최소 1개 이상 탐지되어 있다.

### 실행 절차

1. 사용자가 특정 충돌 후보를 선택한다.
2. GitCat이 AI 호출을 자동 실행한다.
3. 병합 초안 결과를 생성한다.
4. 병합 이유와 해결 방향을 설명하는 중재안을 생성한다.
5. 응답 구조를 Zod 기준으로 검증한다.

### 기대 결과

- AI 병합 초안이 생성된다.
- 중재안이 생성된다.
- 응답 구조가 검증된다.

### 성공 기준

- 병합 초안, 중재안, 응답 검증이 모두 통과하면 성공
