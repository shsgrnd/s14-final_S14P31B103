# S-04

기능: 스냅샷
비고: 최근 스냅샷만 삭제 가능 여부 검증
시나리오명: 가장 최근 스냅샷 수동 삭제
유형: 삭제

### 목적

가장 최근 스냅샷만 수동 삭제 가능한지 확인한다.

### 사전 상태

- 스냅샷이 2개 이상 존재한다.
- 최신 스냅샷과 이전 스냅샷이 목록에서 구분 가능하다.

### 실행 절차

1. 사용자가 스냅샷 목록 화면으로 이동한다.
2. 가장 최근 스냅샷에 삭제 액션이 있는지 확인한다.
3. 이전 스냅샷에 삭제 액션이 없거나 비활성화되었는지 확인한다.
4. 최신 스냅샷 삭제를 실행한다.
5. 확인 모달에서 삭제를 승인한다.

### 기대 결과

- 최신 스냅샷만 삭제 가능하다.
- 삭제 후 목록이 즉시 갱신된다.

### 성공 기준

- 최신 스냅샷만 삭제되고 과거 스냅샷은 삭제되지 않으면 성공
