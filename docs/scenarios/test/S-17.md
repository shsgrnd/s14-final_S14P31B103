# S-17

기능: 브랜치 정리
비고: 자동 감지
시나리오명: 미사용 브랜치 감지
유형: 자동 감지

### 목적

장기간 사용되지 않은 브랜치를 자동으로 감지하여 정리 후보로 제시하는지 확인한다.

### 사전 상태

- 마지막 사용 시점이 오래된 브랜치가 존재한다.

### 실행 절차

1. 사용자가 브랜치 정리 화면으로 이동한다.
2. GitCat이 브랜치 사용 이력을 확인한다.
3. 일정 기간 이상 사용되지 않은 브랜치를 감지한다.
4. 감지된 브랜치를 정리 후보로 표시한다.

### 기대 결과

- 미사용 브랜치가 자동 감지된다.
- 사용자는 이를 정리 후보로 확인할 수 있다.

### 성공 기준

- 미사용 브랜치가 후보 목록에 노출되면 성공
