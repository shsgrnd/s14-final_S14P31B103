# S-15

기능: AI 추천
비고: 추천 결과 생성 및 수정
시나리오명: 커밋 메시지 / 브랜치명 / PR·작업 설명 추천
유형: 정상

### 목적

작업 내용을 기반으로 커밋 메시지, 브랜치명, PR 제목/설명, 작업 설명 추천이 생성되고 수정 가능한지 확인한다.

### 사전 상태

- 변경 파일과 작업 설명이 존재한다.

### 실행 절차

1. 사용자가 추천 기능 화면으로 이동한다.
2. GitCat이 브랜치명을 추천한다.
3. GitCat이 커밋 메시지를 추천한다.
4. GitCat이 PR 제목 및 설명을 추천한다.
5. GitCat이 작업 설명 추천을 제공한다.
6. 사용자가 일부 문구를 수정한다.

### 기대 결과

- 각 추천 항목이 생성된다.
- 사용자가 수정 후 활용할 수 있다.

### 성공 기준

- 추천 생성과 수정 반영이 모두 가능하면 성공
