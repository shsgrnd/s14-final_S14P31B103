# S-02

기능: 스냅샷
비고: 스냅샷 개수 설정 검증
시나리오명: 스냅샷 10개 초과 시 자동 삭제
유형: 경계값

### 목적

스냅샷이 최대 10개만 유지되도록 자동 관리되는지 확인한다.

### 사전 상태

- 기존 스냅샷이 9개 이상 존재한다.

### 실행 절차

1. 사용자가 AI 작업을 반복 실행한다.
2. 스냅샷 수가 10개를 초과하도록 만든다.
3. 스냅샷 목록으로 이동한다.
4. 전체 개수를 확인한다.
5. 가장 오래된 스냅샷이 자동 정리되었는지 확인한다.

### 기대 결과

- 스냅샷 개수는 10개를 넘지 않는다.
- 새 스냅샷이 추가되면 가장 오래된 스냅샷이 자동 삭제된다.

### 성공 기준

- 목록 개수가 항상 10개 이하로 유지되면 성공
