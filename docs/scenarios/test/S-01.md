# S-01

기능: 스냅샷
비고: 2~3초 이내 저장 그루핑 포함
시나리오명: AI 작업 전 자동 스냅샷 정상 생성
유형: 정상

### 목적

AI 작업 실행 직전에 자동 스냅샷이 빠르게 생성되고, 동일 작업 단위 기준으로 그룹핑되는지 확인한다.

### 사전 상태

- VS Code에서 Git 저장소가 열린 상태이다.
- GitCat extension이 활성화되어 있다.
- AI 작업 대상 파일이 존재한다.

### 실행 절차

1. 사용자가 AI 작업 실행 버튼을 선택한다.
2. GitCat이 실행 직전 현재 작업 상태를 수집한다.
3. 자동 스냅샷을 생성한다.
4. 스냅샷 목록 화면으로 이동한다.
5. 새 스냅샷이 생성되었는지 확인한다.
6. 동일 작업 단위 기준으로 그룹핑이 적용되는지 확인한다.

### 기대 결과

- 자동 스냅샷이 생성된다.
- 스냅샷 생성 시간은 2~3초 이내여야 한다.
- 목록에서 작업 단위 기준 그룹핑이 확인된다.

### 성공 기준

- AI 실행 직후 스냅샷이 2~3초 내 생성되고 그룹핑 표시가 보이면 성공
