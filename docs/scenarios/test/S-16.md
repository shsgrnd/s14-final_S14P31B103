# S-16

기능: 브랜치 정리
비고: 삭제 실행
시나리오명: 로컬 브랜치 정리 지원 정상 케이스
유형: 정상

### 목적

사용자가 선택한 로컬 브랜치를 정상적으로 정리할 수 있는지 확인한다.

### 사전 상태

- 삭제 가능한 로컬 브랜치가 존재한다.

### 실행 절차

1. 사용자가 브랜치 정리 화면으로 이동한다.
2. 브랜치 목록을 확인한다.
3. 삭제 대상 브랜치를 선택한다.
4. 삭제 실행 버튼을 선택한다.
5. 확인 모달을 승인한다.

### 기대 결과

- 선택한 브랜치가 삭제된다.
- 결과 메시지가 표시된다.

### 성공 기준

- 삭제 대상 브랜치가 목록에서 제거되면 성공
