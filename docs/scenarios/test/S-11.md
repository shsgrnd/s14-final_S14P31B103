# S-11

기능: 컨텍스트
비고: 스냅샷 연동
시나리오명: 변경 기록 저장 및 컨텍스트 구성
유형: 정상

### 목적

변경 기록이 저장되고, 해당 기록이 스냅샷과 연동되어 AI 제안의 컨텍스트로 사용되는지 확인한다.

### 사전 상태

- 사용자가 AI 작업 또는 병합 관련 작업을 수행한 상태이다.

### 실행 절차

1. 작업 완료 후 GitCat이 변경 파일 목록을 수집한다.
2. 수정 위치와 작업 설명을 함께 저장한다.
3. 변경 기록이 스냅샷과 연결되는지 확인한다.
4. 이후 AI 제안 생성 시 저장된 기록이 참조되는지 확인한다.

### 기대 결과

- 변경 기록이 저장된다.
- 스냅샷과 연결된다.
- 이후 AI 제안에서 해당 기록이 컨텍스트로 사용된다.

### 성공 기준

- 변경 기록 조회 가능, 스냅샷 연동, AI 활용까지 확인되면 성공
