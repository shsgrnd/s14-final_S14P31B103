# S-03

기능: 스냅샷
비고: 즐겨찾기 시각 표시 검증
시나리오명: 스냅샷 즐겨찾기 표시 확인
유형: UI 표시

### 목적

즐겨찾기 설정 시 사용자가 시각적으로 구분 가능한 표시를 확인할 수 있는지 검증한다.

### 사전 상태

- 스냅샷이 2개 이상 존재한다.

### 실행 절차

1. 사용자가 특정 스냅샷을 선택한다.
2. 즐겨찾기 아이콘 또는 별표 버튼을 선택한다.
3. 목록 화면에서 해당 스냅샷을 다시 확인한다.

### 기대 결과

- 즐겨찾기한 스냅샷에 별표 또는 컬러 강조가 표시된다.
- 일반 스냅샷과 구분된다.

### 성공 기준

- 즐겨찾기 여부가 목록에서 즉시 식별 가능하면 성공
