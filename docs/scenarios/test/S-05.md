# S-05

기능: 원복
비고: 선택한 과거 스냅샷으로 복원
시나리오명: AI 작업 단위 원복 정상 케이스
유형: 정상

### 목적

선택한 과거 스냅샷 기준으로 파일 상태를 정상 복원할 수 있는지 확인한다.

### 사전 상태

- AI 작업으로 인해 코드가 변경된 상태이다.
- 원복 가능한 과거 스냅샷이 존재한다.

### 실행 절차

1. 사용자가 원복 화면으로 이동한다.
2. 과거 스냅샷을 선택한다.
3. 복원 대상 파일과 변경 시점을 확인한다.
4. 원복 실행을 선택한다.

### 기대 결과

- 선택한 과거 스냅샷 시점으로 복원된다.
- 복원 완료 메시지가 표시된다.

### 성공 기준

- 파일 상태가 선택한 스냅샷과 일치하면 성공
