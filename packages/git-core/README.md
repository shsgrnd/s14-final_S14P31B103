# GitCat Git Core

이 패키지는 GitCat이 제공하는 핵심 가치 중 "로컬 파일 관리" 및 "Git 제어"를 담당하는 모듈입니다.
Git 명령어를 감싸서 애플리케이션 내에서 쉽게 사용할 수 있는 함수나 클래스 형태로 제공합니다.

## 주요 기능
- **`src/snapshot/`**: AI 작업을 수행하기 직전에 워크트리를 백업하고, 문제 발생 시 즉각 롤백할 수 있는 스냅샷 생성/복원 로직
- **`src/diff/`**: 충돌이 난 파일이나 변경된 사항을 diff 형태나 patch 형태로 파싱하고 추출하는 로직
- **`src/branch/`**: 로컬 브랜치 목록 조회, 정리, 브랜치명 자동 추천에 쓰이는 기반 로직
